
==============================
K-MEANS CLUSTERING — CLEAN NOTES
==============================

1) WHAT IS K-MEANS?
------------------------------
K-means is an unsupervised clustering algorithm that groups data points
into K clusters by minimizing the Within-Cluster Sum of Squares (WCSS).

• Type: Unsupervised
• Output: Cluster labels + centroids
• Assumes spherical, equally sized clusters

------------------------------
2) GOAL OF K-MEANS
------------------------------
• Similar points → same cluster
• Dissimilar points → different clusters
• Minimize distance between points and their cluster centroid

------------------------------
3) IMPORTANT TERMS
------------------------------
Centroid:
• Mean of all points in a cluster

WCSS:
WCSS = Σ || x - μ ||²
• Measures cluster compactness
• Lower WCSS → tighter clusters

------------------------------
4) COMPLETE WORKFLOW (STEP-BY-STEP)
------------------------------

STEP 0: Choose K (number of clusters)
• Domain knowledge OR
• Elbow Method

STEP 1: Initialize Centroids
• Randomly select K points as initial centroids
• One centroid per cluster

STEP 2: Assignment Step
• For each data point:
  - Compute distance to all centroids
  - Assign point to nearest centroid

STEP 3: Update Step
• For each cluster:
  - Compute new centroid = mean of assigned points

STEP 4: Repeat
• Repeat Assignment + Update
• Until:
  - Centroids stop changing OR
  - Max iterations reached

------------------------------
5) FINAL OUTPUT
------------------------------
• Cluster label for each point
• Final centroids
• Final WCSS

------------------------------
6) ELBOW METHOD (SELECTING K)
------------------------------
• Try K = 1,2,3,...
• Run full K-means for each K
• Compute WCSS for each K
• Plot: K vs WCSS
• Choose K where decrease slows sharply (elbow)

------------------------------
7) WHY WCSS ALWAYS DECREASES
------------------------------
• More clusters → fewer points per cluster
• Extreme case:
  K = number of points → WCSS = 0

------------------------------
8) LIMITATIONS OF K-MEANS
------------------------------
• Need to choose K beforehand
• Sensitive to initialization
• Fails on non-spherical clusters
• Sensitive to outliers

------------------------------
9) IMPROVEMENTS
------------------------------
• K-means++ (better initialization)
• Standardization before clustering

------------------------------
MEMORY HOOK
------------------------------
Choose K → Assign → Mean → Repeat
Elbow chooses K, not clusters
